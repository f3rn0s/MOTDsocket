Welcome to MOTDsocket

This is a simple one man project to making a MOTD server that can give people information, much like an RSS feed. It is currently operational, but in VERY early development.

Current Features

Uses file motd.txt to relay message to user. Uses threading for multiple users. Outputs connected IP's to a file.

Authors and Contributors

Tealkine (@tealkine) is the only contributor currently, and it will probably stay that way.

Planned Development

Host multiple servers in one python server; e.g. - Join with port 50, get MOTD, Join with port 31337, get information etc.
